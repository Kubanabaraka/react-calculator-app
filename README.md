# React Calculator App

Simple calculator built with React, TypeScript, and Vite.

## Prerequisites

- Node.js 18+
- npm

## Quick Start

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start development server
- `npm run build` - create production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
